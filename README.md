# AI-Powered Intelligent Transport System (ITS) – Computer Vision

This repository contains my Computer Science Honours research prototype for an AI-powered intelligent transport system designed to improve pothole reporting and municipal monitoring.

## Project Overview

The system integrates:
- a Flutter mobile application
- computer vision (YOLOv8) for pothole detection
- TensorFlow Lite for mobile inference
- Firebase for backend services
- GPS-based reporting
- a municipality dashboard workflow

## Key Features

- User registration and login
- Pothole reporting via mobile app
- AI-based pothole detection
- GPS-linked reporting
- Viewing reported potholes
- Dashboard interface for municipal monitoring

## Technologies

- Python
- YOLOv8
- TensorFlow Lite
- Flutter / Dart
- Firebase
- OpenCV
- Google Maps API

## Structure

- `mobile_app/` → Flutter prototype code
- `screenshots/` → App, dashboard, and model outputs
- `docs/` → project notes
- `notes/` → implementation notes

## Important Note

This project represents an honours research prototype.  
Some parts of the application are incomplete or may not run fully due to later changes during development.

Screenshots represent real outputs and UI flows captured during the project.
